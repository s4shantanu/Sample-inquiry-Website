# aid
## For Responsiveness i used this code in assets/css/style.css
```css

.blg {
    visibility: show;
    width: 50%;
    
}

.bls {
    width: 50%;
}

@media only screen and (max-width:700px) {
    .blg {
        display: none;
        width: 0%;
    }
    .bls {
        width: 100%;
    }
}
```
#### and please follow this structure
##### and for changing placeholder color use
```css
::placeholder { 
    color:#fff;
    }
```
